# ms-architecture-icons
SVG icons for architecture diagrams of Microsoft services

Microsoft offers a fantastic collection of different architectural icons for creating architecture diagrams, but unfortunately these icons are not currently availabe in Mermaid.js. 

For this reason I have compiled the following SVG collections in usage in Mermaid.js:

- https://learn.microsoft.com/en-us/azure/architecture/icons/ (v23)

I also intend to the add the following in future:

- https://learn.microsoft.com/en-us/microsoft-365/solutions/architecture-icons-templates?view=o365-worldwide
- https://learn.microsoft.com/en-us/dynamics365/get-started/icons
- https://learn.microsoft.com/en-us/power-platform/guidance/icons
- https://learn.microsoft.com/en-us/entra/architecture/architecture-icons

## Microsoft's Usage Guidelines

Do's
- Use the icons to illustrate how products can work together
- In diagrams, we recommend to include the product name somewhere close to the icon
- Use the icons as they would appear within Azure

Don'ts
- Don't crop, flip or rotate icons
- Don't distort or change icon shape in any way
- Don't use Microsoft product icons to represent your product or service
